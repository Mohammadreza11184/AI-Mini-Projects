#AI project description
1. Head-Controlled Mouse
Controls the computer mouse cursor using head movements tracked via webcam (OpenCV).

2. License Plate Recognition & Segmentation
Detects vehicle license plates and segments the digits/characters for OCR (Optical Character Recognition) using AI.

3. Cancer Detection
Analyzes medical images to detect cancerous cells and tumors using deep learning (CNN).

4. Fruit Sorting
Classifies and sorts different types of fruits based on visual features like color, size, and texture using computer vision.

5. Face Recognition with Naming
Recognizes and compares faces to identify known individuals, then displays their name on top of the bounding box in real-time.

6. Bitcoin Price Prediction
Predicts future Bitcoin prices using time-series forecasting

7. Bank Loan Eligibility Prediction
Predicts whether a person qualifies for a bank loan based on financial and personal data (credit score, income, etc.).
